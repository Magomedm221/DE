## GitHub
- Прочитать про GitHub

- Для управления GitHub использовать коммандную строку

Сам код: 
```
- git clone
- git switch -c new_branch
- *редактируем, пишем, созидаем*

- git commit -m "краткое описание коммита"
- git checkout main
- git merge new_branch
- git push
```
